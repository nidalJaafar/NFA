Darsa NFA

    Operations:
  • \(\mid\) union
  • \(^*\) kleene star operator
  • Concatenation is done by putting letters beside each other
  • \(()\) for precedence

Examples:
\((a\mid b)^*\)
First we do \(a\) union \(b\) then we star the result
\((ab\mid b)^*\)
First \(b^*\) then \(ab\) gets concatenated then \(b^*\) union \(ab\)
Note
Input should consist of small letters from \(a\) to \(z\) except the letters \(o\) and \(x\)
